# 入库接口

  • 站外入库接口一般是给第三方软件（如：火车头、ET等）提供的入库方案。
  • 使用前请在后台系统-站外入库配置 设置好 免登录入库密码，和 分类名称对应系统分类规则。
  • 系统默认api入库接口文件为http://域名/api.php，根据需要可修改入库文件。

# 下载模块

官方提供的 【苹果CMSV10火车头入库模块】 (opens new window)！

# 火车头配置

网站根地址填写 http://域名/api.php/点击测试，输入验证密码，何必要信息测试入库。

接口必要信息是

名称， 分类ID 或 分类名称 （如果同时存在以 分类ID为准）

# 入库接口地址

  • 视频 /api.php/receive/vod
  • 文章 /api.php/receive/art
  • 演员 /api.php/receive/actor
  • 角色 /api.php/receive/role
  • 网址 /api.php/receive/website

# 接口参数

  • 请求方式 POST
  • Content-Type form-data

可以是模型的全字段，如视频入库可以使用 视频字段 全部字段作为参数，

通用必填参数

参数 示例值 是否必填 参数描述
pass a54s848s 必填 入库密码，后台可自定义
type_id 1 必填 绑定分类id
XXX_name 极地救援预告片 必填 名称必填 vod_name、art_name、actor_name、vod_role

视频必填

参数 示例值 是否必填 参数描述
vod_play_from dplayer 必填 后台播放器id ，空值会发布失败
vod_play_url 高清$http://www.baidu.com/vod.mp4 必填 播放地址，没有播放地址会发布失败

多个播放组可是使用 $$$ 进行分割id，然后播放地址也要进行分组，也是用 $$$ 如下示例:

<!-- 播放器分组 -->
dplayer$$$videojs$$$ppvod
<!-- 播放地址分组 -->

高清$http://www.1.com/vod.mp4#高清$http://www.1.com/vod.mp4#高清$http://www.1.com/vod.mp4$$$

# 开放资源api

api接口仅供提供数据，可以直接在苹果CMS后台联盟采集中加入-并提供给他人采集。 联盟资源分配唯一标识ID，用来区别绑定分类，这个ID一般由苹果CMS官方提供，不可随意 修改设置，否则造成入库分类错乱。

TIP

视频接口同时支持老板xml格式的数据，增加参数 &at=xml即可。

# 视频列表地址

  • 接口地址 /api.php/provide/vod/?ac=list

# 视频详情地址

  • 接口地址 /api.php/provide/vod/?ac=detail

# 文章列表地址

  • 接口地址 /api.php/provide/art/?ac=list

# 文章详情地址

  • 接口地址 /api.php/provide/art/?ac=detail
