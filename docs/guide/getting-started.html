# 快速上手

# 安装环境可以使用宝塔面板 (opens new window) 运行环境，以下是使用宝塔面板创建苹果cms站长详细步骤教程

前提条件

  • Windows 平台： IIS/Apache/Nginx/等 + PHP(5.5 +) + MySQL(5.5 +)
  • Linux/Unix 平台： Apache/Nginx/等 + PHP (5.5 +) + MySQL(5.5 +) (PHP必须在非安全模式下运行)

# 添加网站

  • 绑定域名
  • 创建mysql数据库
  • 选择PHP版本为 5.6以上

# 下载安装包

  • 进入添加好的网目录

  • 点击远程下载，输入
https://cdn.jsdelivr.net/gh/magicblack/maccms_down@master/maccms10.zip
1

点击确认，

  • 下载完成后在线解压

  • 解压完成后打开网站域名就可以进入安装界面：【进入安装步骤】

也可以到 到 github 下载安装包，地址为：

https://github.com/magicblack/maccms_down/
1

然后上传到创建的网站并在线解压安装包

还可以Git拉取苹果cmsv10最新源码 我的安装目录在E盘的苹果cmsv10目录 输入

$ cd E:/苹果cmsv10
$ git init
$ git remote add origin https://github.com/magicblack/maccms10.git  
$ git pull origin master
1
2
3
4

稍等片刻 所有源码就被下载到 E盘的 苹果cmsv10了

# 进入安装步骤

  • 上传安装包到网站根目录后，访问站点就进入安装界面了
  • 输入你创建的数据库名 用户名和密码
  • 点击测试连接无误后进行下一步
  • 配置后台管理员账号和密码

注意

新建空网站最好选择数据覆盖和初始默认数据

安装完成后回提示你 修改后台默认入口文件名 admin.php,进入网站根目录下找到 admin.php 重命名为别的就可以，例如xxxx-admin.php 然后 浏览器输入你的域名+自定义后面入库文件即可进入苹果cms管理后台 如：http://www.datll.com/xxxx-admin.php
