          # 行業術語

          你可能會在文檔中碰到一些陌生的概念，本節列出了文檔中常見的術語，方便使用、學習、插件/主題開發之用。

          # 資源站

          就是專門給其他網站提供數據采集的站點，站長可以在後臺直接對接資源站的api，實時更新網站內容。

          # 視頻解析

          視頻解析指的是對正版視頻的盜版盜播行為，並且能夠去除廣告和試看時間限製，搭建解析和使用解析都是侵權違法行為，大家請不要以身試法。

          • 解析接口 以提交url作為參數的解析api
          • 官解 官方視頻解析【
          • 二次解析 對解析接口進行解析

          # 采集規則

          一般指的是爬蟲軟件【火車頭、八爪魚】的爬數據的規則，這軟件需要對目標站進行采集是需要配置符合該網站的爬取規則。

          # CF

          這裏的CF可不是騰訊遊戲裏的cf哦。而一個國外的免費網站加速和防護工具，全稱 CloudFlare 官網是：https://www.cloudflare.com/zh-cn/。 CloudFlare Web應用防火墻是一種智能、集成且可擴展的解決方案，可保護您的關鍵業務 Web 應用程序免受惡意攻擊，並且無需更改現有基礎架構。

          # 假墻攻擊

          最近有一幫專業做黑產的團隊，以刷站在統計通知你“你的網站即將被攔截、我們即將對你網站進行攻擊、您的網站將在某某時間停止訪問”等等勒索的留言，其目的是為了讓你跟他們合作並網站掛他們的非法廣告，如果你拒接或者不理會，他們會對你請求攜帶惡意、敏感、涉及政治的關鍵詞，導致你網站被國家防火墻臨時攔截，使你的網站在國內無法訪問。

          # 假墻攻擊的原理：

          • 1，他們一般先攻擊dns服務（造成部分地區無法訪問，感覺和被墻一樣。） 說白了就是DNS攻擊。 【解決辦法可以買個 dns.com解析專業版 1年才不到200塊 ，就不用在輪詢cf了】
          • 2，然後長期攻擊違規關鍵字（搜索頁和篩選頁） 會被搜索引擎收錄。 導致最終被真墻~~~~【解決辦法升級到最新版本開啟假墻防禦】
          • 3，如果你的站已經打的 搜索引擎搜索了違規詞， 那麽趕緊換域名把，開啟防禦也沒效果。 如果沒有收錄到，那麽恭喜你 有救。
