蘋果cmsV10

基於ThinkPHP和Layui的多功能開源免費內容管理系統

Get Started →

簡單易用

大幅度降低站長上手門檻，讓每一個無需會編程技術的新手都能快速上手。這將會是你建站學習的最佳cms。

模板標簽

系統內置了豐富的cms標簽並支持thinkphp框架標簽完美融合，可以調取系統內所有的數據，只要你懂一些前端知識就可以快速的開發自己的模板功能。

後臺管理

管理後臺界面大方、操作簡單、功能齊全、模塊眾多、雙端管理，只要你會電腦或手機就會使用。

豐富模塊

支持視頻模塊、文章模塊、專題模塊、網址導航模塊、明星模塊、劇情模塊、角色模塊、會員模塊。

文檔豐富

蘋果cmsV10提供更全面、更詳細、更專業的開源cms技術文檔，從而更快速地解決疑難問題

插件擴展

靈活強大應用插件系統為你解決二次開發所面臨的升級問題，享受擴展php功能同時還支持cms官方升級。

# git命令獲取最新倉庫源碼

$ cd E:/maccms
$ git init
$ git remote add origin https://github.com/magicblack/maccms10.git  
$ git pull origin master
