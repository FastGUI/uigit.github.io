# 入庫接口

  • 站外入庫接口一般是給第三方軟件（如：火車頭、ET等）提供的入庫方案。
  • 使用前請在後臺系統-站外入庫配置 設置好 免登錄入庫密碼，和 分類名稱對應系統分類規則。
  • 系統默認api入庫接口文件為http://域名/api.php，根據需要可修改入庫文件。

# 下載模塊

官方提供的 【蘋果CMSV10火車頭入庫模塊】 (opens new window)！

# 火車頭配置

網站根地址填寫 http://域名/api.php/點擊測試，輸入驗證密碼，何必要信息測試入庫。

接口必要信息是

名稱， 分類ID 或 分類名稱 （如果同時存在以 分類ID為準）

# 入庫接口地址

  • 視頻 /api.php/receive/vod
  • 文章 /api.php/receive/art
  • 演員 /api.php/receive/actor
  • 角色 /api.php/receive/role
  • 網址 /api.php/receive/website

# 接口參數

  • 請求方式 POST
  • Content-Type form-data

可以是模型的全字段，如視頻入庫可以使用 視頻字段 全部字段作為參數，

通用必填參數

參數 示例值 是否必填 參數描述
pass a54s848s 必填 入庫密碼，後臺可自定義
type_id 1 必填 綁定分類id
XXX_name 極地救援預告片 必填 名稱必填 vod_name、art_name、actor_name、vod_role

視頻必填

參數 示例值 是否必填 參數描述
vod_play_from dplayer 必填 後臺播放器id ，空值會發布失敗
vod_play_url 高清$http://www.baidu.com/vod.mp4 必填 播放地址，沒有播放地址會發布失敗

多個播放組可是使用 $$$ 進行分割id，然後播放地址也要進行分組，也是用 $$$ 如下示例:

# 開放資源api

api接口僅供提供數據，可以直接在蘋果CMS後臺聯盟采集中加入-並提供給他人采集。 聯盟資源分配唯一標識ID，用來區別綁定分類，這個ID一般由蘋果CMS官方提供，不可隨意 修改設置，否則造成入庫分類錯亂。

TIP

視頻接口同時支持老板xml格式的數據，增加參數 &at=xml即可。

# 視頻列表地址

  • 接口地址 /api.php/provide/vod/?ac=list

# 視頻詳情地址

  • 接口地址 /api.php/provide/vod/?ac=detail

# 文章列表地址

  • 接口地址 /api.php/provide/art/?ac=list

# 文章詳情地址

  • 接口地址 /api.php/provide/art/?ac=detail
