特色

v10采用 ThinkPHP5.x (opens new window)內核進行開發，擴展了模板處理引擎，將後臺程序與html模板簡單的分離出來，讓設計人員與程序人員最大限度的發揮自己的優勢而互不幹擾，大大加快了項目有序、快速的完成。 即使您是第一次接觸，也會在最短的時間內熟練掌握它的使用方法。後臺管理模塊，一目了然，操作簡單，絕對不會讓您眼花繚亂。

# 程序介紹

蘋果CMS程序是一套采用PHP+MYSQL環境下運行的完善而強大的快速建站系統。 經過近多年的開發經驗和技術積累，蘋果CMS程序已逐步走向成熟，在易用性和功能上已經成為同行中的佼佼者。 程序體積小->優化程序代碼，運行速度快->高效的緩存處理，只要普通的虛擬主機就可以完美搭建起來，建站成本非常低。仿MVC模板分離，內置標簽，自定義函數標簽接口，強大的自定義采集功能，只要你會HTML就可以輕松做出個性化的網站。 程序易用性和功能上一直以來都積極采納廣大站長提出的各種好的建議，迅速響應各種緊急問題，我們的服務理念貫穿其中，保證每一位站長每一個環節都可以從容應對。

# 蘋果CMS能做什麽？

蘋果CMS功能非常強大，模塊豐富。可以用於搭建專業的視頻站如：收費網課類、國際教師視頻站，也可以用於搭建群站、文章資訊站、圖片素材站、網址導航站等等類型，還可以擴展開發搭建移動APP的服務端。蘋果cms的模板豐富，網絡上有海量的免費模板可以使用，無需編程基礎即可極速搭建網站。
