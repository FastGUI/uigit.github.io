    # 2020.1000.1060

    • 修复生成下载页的问题。

    • 修复播放器导入跨站攻击的问题。
    • 防止上传后门关闭应用中心上传本地插件功能。
    • 优化后台登录验证。
    • 修复同步图片显示链接的问题。
    • 修复后台管理留言搜索的问题。
    • 修复词库随机插入语句的问题。
    • 修复发送邮件问题。
    • 用户中心上传头像支持base64编码方式参数名imgdata。
    • 其他细节。

    • 后台+安装界面提示信息支持多简繁体切换。
    • 请参考多语言说明文档可自定义显示信息。
    • 其他细节。

    # 2020.1000.1042

    • 新增数据库挂马检测功能。
    • 新增评论入库接口。
    • 角色入库接口支持豆瓣id对应视频。
    • 定时任务可以强制执行。
    • 支持自定义404页面模板。
    • 后台主要操作内容加入token限制，防止跨站攻击。
    • 模板编辑保存禁止出现更多关键字。
    • 其他细节。
