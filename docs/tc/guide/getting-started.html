# 快速上手

# 安裝環境可以使用寶塔面板 (opens new window) 運行環境，以下是使用寶塔面板創建蘋果cms站長詳細步驟教程

前提條件

  • Windows 平臺： IIS/Apache/Nginx/等 + PHP(5.5 +) + MySQL(5.5 +)
  • Linux/Unix 平臺： Apache/Nginx/等 + PHP (5.5 +) + MySQL(5.5 +) (PHP必須在非安全模式下運行)

# 添加網站

  • 綁定域名
  • 創建mysql數據庫
  • 選擇PHP版本為 5.6以上

# 下載安裝包

  • 進入添加好的網目錄

  • 點擊遠程下載，輸入
https://cdn.jsdelivr.net/gh/magicblack/maccms_down@master/maccms10.zip
1

點擊確認，

  • 下載完成後在線解壓

  • 解壓完成後打開網站域名就可以進入安裝界面：【進入安裝步驟】

也可以到 到 github 下載安裝包，地址為：

https://github.com/magicblack/maccms_down/
1

然後上傳到創建的網站並在線解壓安裝包

還可以Git拉取蘋果cmsv10最新源碼 我的安裝目錄在E盤的蘋果cmsv10目錄 輸入

$ cd E:/蘋果cmsv10
$ git clone https://github.com/magicblack/maccms10.git
$ cd maccms10
1
2
3

稍等片刻 所有源碼就被下載到 E盤的 蘋果cmsv10的 maccms10了

# 進入安裝步驟

  • 上傳安裝包到網站根目錄後，訪問站點就進入安裝界面了
  • 輸入你創建的數據庫名 用戶名和密碼
  • 點擊測試連接無誤後進行下一步
  • 配置後臺管理員賬號和密碼

註意

新建空網站最好選擇數據覆蓋和初始默認數據

安裝完成後回提示你 修改後臺默認入口文件名 admin.php,進入網站根目錄下找到 admin.php 重命名為別的就可以，例如xxxx-admin.php 然後 瀏覽器輸入你的域名+自定義後面入庫文件即可進入蘋果cms管理後臺 如：http://www.datll.com/xxxx-admin.php
