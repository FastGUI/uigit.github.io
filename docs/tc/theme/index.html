# 模板

什麽是模板

模板就是網站的主題外觀，又稱主題、皮膚，使用不同的模板可以使網站前臺以不同的樣式展示，如同人們的衣著打扮一樣，會穿著會打扮，人肯定比較漂亮，網站也是如此，一個好看的模板不僅僅能吸引用戶的第一視覺，還能把網站功能以更加人性化地的展示給用戶。
