# 剧情

剧情是唯一没有自己独立标签的模块，因此调用剧情的方式和调用视频一样都是用 maccms:vod 标签进行。

  • 目录 html/plot
  • 所属模型 model('Plot')
  • 模型mid 10，使用{$maccms.mid}标签即可获得当前模型mid值

# 剧情首页

  • 文件目录 html/plot/index.html
  • 页面aid值 100

# 剧情搜索

  • 文件目录 html/plot/search.html
  • 页面aid值 103

# 剧情详情

  • 文件目录 html/plot/detail.html
  • 页面aid值 104
