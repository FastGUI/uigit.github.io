    # 模型

    苹果cms 插件中的模型使用方法完全同ThinkPHP5的模型使用相同，请直接参考ThinkPHP5官方文档模型章节。

    请参考：ThinkPHP5模型 (opens new window)

    特别提醒

    特别注意下命名空间的使用。

    # 如果你的模型存放位置是

    addons/mydemo/model/User.php
    1

    # 则你的模型的命名空间为

    addon\mydemo\model
    1

    # 正确的使用方法如下：

    namespace addons\mydemo\model;
use think\Model;
class User extends Model
{
    
}
