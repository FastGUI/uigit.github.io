# 蘋果cmsV10整合雲轉碼（PPVOD）插件 (opens new window)

(opens new window)

蘋果cms下載 (opens new window) | PPVOD（雲轉碼）下載 (opens new window) | 插件官方 (opens new window)

# 介紹

特色

該插件不會替換系統任何文件，可關閉、可卸載不殘留文件、不影響cms版本升級。使用方法簡單。內置防盜鏈、多碼率播放器， 支持離線下載到PPVOD，一鍵復製分享地址、m3u8地址、遠程刪除PPVOD數據、一鍵入庫功能、支持從cms後臺上傳影片到ppvod並返回m3u8地址。

# 安裝教程：

  1. 將插件文件夾裏的 ppvod 文件夾上傳到 蘋果cms 網站 的addons 目錄，然後到後臺 > 應用 > 應用市場 > ppvod轉碼插件 > 啟用插件就可

  2. 插件配置 填寫正確的 ppvod 對接地址 、api密鑰 保存 配置並刷新後臺！

  3. 在"應用"菜單下就會多出來一個 PPVOD 選項

# 播放器配置：

1.在插件啟用後會在 蘋果cmsV10後臺 “視頻” => “播放器” 列表末端新增一個 “PPVOD” 的播放器， 2.該播放器支持 防盜鏈和蘋果cmsV10自帶試看權限配置，可直接使用該播放器無需其他配置

# 自動推送：

1.正常配置插件 並 啟用後，在蘋果cms 後臺 》 應用 》 ppvod 菜單下， 點擊 “推送地址” 自動復製拼接好的 推送地址 到剪切板；

2.然後到 PPVOD管理後臺 》 系統設置 》基礎設置 》 推送地址 進行粘貼剪切板的地址

# 常見問題

# 什麽是雲轉碼接口

答案：就是你PPVOD的後臺地址，如：http://123.144.12.215:2000

# 什麽是播放域名？

答案：您單獨配置的播放域名，如果沒有，默認為 雲轉碼地址+2100端口，如：http://123.144.12.215:2000

# 什麽是上傳密鑰？

答案：當前版本不要填寫

# 什麽是讀寫密鑰？

答案：用於蘋果cms和ppvod的通訊以及讀寫權限。

# 什麽是防盜密鑰？

答案：用於配置防盜播功能，可現在 ppvod 系統設置 -> 防盜系統 裏先配置好密鑰，然後在吧密鑰復製到這裏填寫。

# 什麽是推送密碼？

答案：無需從其他地方獲取，這裏就是定義密碼的地方,隨意填寫即可！ 推送的完整URL可以在你啟用ppvod插件並且設置完密碼後到 蘋果cms 後臺-> 應用 -> PPVOD -> 推送地址【點擊一下自動拷貝到剪貼板】，然後在到ppvod基礎設置裏的推送地址粘貼即可。

# 我怎麽推送？

  • 勾選所需內容，點擊推送按鈕即可，自動把勾選的內容推送到網站，如果你在軟件後臺的分類名字和網站的分類名字相同，還能自動根據網站分類對應入庫。是不是很方便？
  • 使用此功能，首先需要在PPVOD後臺-基礎設置-推送地址，填上你網站的推送插件地址。需要註意的是推送地址的域名不能使用cdn域名，如果網站必須使用cdn，必須另外綁定一個域名（網站服務器的ip也可以）到網站，推送地址就用這個沒有經過cdn的域名。
  • 為了防止惡意入庫，一般都要攜帶入庫密碼，這個入庫密碼就是你的推送密碼？

# 什麽是附件地址？

答案：就是你ppovd視頻封面的地址，有的站長單獨配置了域名，如果沒有另外設置圖片域名這裏就填寫 ppvod 的地址就可以了。

# 請求解析服務器失敗？

答案：如果配置無誤，有可能SSL問題，就是你蘋果cms後臺使用https協議，但是PPVOD接口是為http導致的通訊失敗，只要保持兩個地方協議一致即可解決。

升級須知

無論是cms升級還是插件升級都必須先禁用插件，然後在覆蓋文件，覆蓋完成後在點擊啟用插件，不然禁用或啟用插件時會提示文件沖突。
