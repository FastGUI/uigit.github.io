          # 行业术语

          你可能会在文档中碰到一些陌生的概念，本节列出了文档中常见的术语，方便使用、学习、插件/主题开发之用。

          # 资源站

          就是专门给其他网站提供数据采集的站点，站长可以在后台直接对接资源站的api，实时更新网站内容。

          # 视频解析

          视频解析指的是对正版视频的盗版盗播行为，并且能够去除广告和试看时间限制，搭建解析和使用解析都是侵权违法行为，大家请不要以身试法。

          • 解析接口 以提交url作为参数的解析api
          • 官解 官方视频解析【
          • 二次解析 对解析接口进行解析

          # 采集规则

          一般指的是爬虫软件【火车头、八爪鱼】的爬数据的规则，这软件需要对目标站进行采集是需要配置符合该网站的爬取规则。

          # CF

          这里的CF可不是腾讯游戏里的cf哦。而一个国外的免费网站加速和防护工具，全称 CloudFlare 官网是：https://www.cloudflare.com/zh-cn/。 CloudFlare Web应用防火墙是一种智能、集成且可扩展的解决方案，可保护您的关键业务 Web 应用程序免受恶意攻击，并且无需更改现有基础架构。

          # 假墙攻击

          最近有一帮专业做黑产的团队，以刷站在统计通知你“你的网站即将被拦截、我们即将对你网站进行攻击、您的网站将在某某时间停止访问”等等勒索的留言，其目的是为了让你跟他们合作并网站挂他们的非法广告，如果你拒接或者不理会，他们会对你请求携带恶意、敏感、涉及政治的关键词，导致你网站被国家防火墙临时拦截，使你的网站在国内无法访问。

          # 假墙攻击的原理：

          • 1，他们一般先攻击dns服务（造成部分地区无法访问，感觉和被墙一样。） 这几天maccms.la一样被攻击，效果基本一致。 【解决办法可以买个 dns.com解析专业版 1年才不到200块 ，就不用在轮询cf了】
          • 2，然后长期攻击违规关键字（搜索页和筛选页） 会被搜索引擎收录。 导致最终被真墙~~~~【解决办法升级到最新版本开启假墙防御】
          • 3，如果你的站已经打的 搜索引擎搜索了违规词， 那么赶紧换域名把，开启防御也没效果。 如果没有收录到，那么恭喜你 有救。
